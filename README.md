# Titanic Machine Learning

This repository contains a machine learning project focused on predicting the survival of passengers on the Titanic.

## Project Structure

- `data/`: Contains the dataset used for analysis.
- `notebooks/`: Jupyter notebooks with exploratory data analysis and model training.
- `src/`: Source code for data processing and model evaluation.
